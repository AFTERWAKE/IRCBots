artBot
======

A simple IRC bot that sends out ASCII messages on the Art channel.

Dependencies
------------

* Twisted (python)

Config
------

* Nick: Nickname of the bot
* Admin: Administrator of the bot
* Channel: Server channel
* Server: URL that the bot connects to
