# Magic Conch

## Dependencie
- twistedIRC
- lxml

## Features
- Helps you make decisions
- Tell you where you should go
- Answer any other questions you might have
- Be hailed
- Choose a Pokemon

## Config
Conch recognizes her admin via textfile "admin_ip.txt"
This file needs to be located in the base directory

## Admin Commands
- ignore <nick>
- unignore <nick>
- say <.*>
